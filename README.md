Overview

Secure180 is a collaborative project built by me and my friend.
The main aim of this project is to understand and implement basic security principles in web development. It focuses on building awareness about common vulnerabilities and how to prevent them using secure coding practices.

Features
Secure login and authentication system
Input validation to prevent malicious attacks
Protection against common vulnerabilities like XSS and SQL Injection
Simple and clean project structure
Easy to understand and extend for future improvements
Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js / Express (if used)
Database: MongoDB / MySQL (if used)
Tools: Git, GitHub
